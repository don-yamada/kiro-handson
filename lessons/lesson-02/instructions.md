## 💡 効率的な指示のコツ

### 明確で具体的に

❌ 悪い例:
```
コードを書いて
```

✅ 良い例:
```
src/app.jsにTodoを追加する関数を作成してください。
関数名はaddTodoで、引数はtodoTextです。
```

### 一度に1つのことを

❌ 悪い例:
```
ファイルを作成して、関数を追加して、テストも書いて、ドキュメントも更新して
```

✅ 良い例:
```
まず、src/app.jsを作成してください。
```
（完了後）
```
次に、addTodo関数を追加してください。
```

### 期待する結果を伝える

❌ 悪い例:
```
リファクタリングして
```

✅ 良い例:
```
この関数をリファクタリングして、初心者でも理解しやすいように、
変数名を説明的にして、コメントを追加してください。
```
