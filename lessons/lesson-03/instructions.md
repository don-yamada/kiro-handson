## Hookの作成

### パターン1: README自動更新

```
AgentHooksを設定して、JavaScriptファイルを保存したときに、
README.mdの「最終更新日時」を自動で更新してください。
```

### パターン2: ファイルヘッダー自動追加

```
AgentHooksを設定して、新しいJavaScriptファイルを作成したときに、
ファイルの先頭に以下の情報を自動で追加してください：
- ファイル名
- 作成日時
- 簡単な説明
```

### パターン3: コードフォーマット

```
AgentHooksを設定して、JavaScriptファイルを保存したときに、
自動でコードをフォーマットしてください。
インデントは2スペースにしてください。
```

### パターン4: コメント自動追加

```
AgentHooksを設定して、関数を追加したときに、
自動でJSDocコメントを生成してください。
```

### パターン5: エラーチェック

```
AgentHooksを設定して、JavaScriptファイルを保存したときに、
構文エラーがないか自動でチェックしてください。
エラーがあれば、修正案を提示してください。
```

### パターン6: テスト自動実行

```
AgentHooksを設定して、テストファイル（*.test.js）を保存したときに、
自動でテストを実行して、結果を報告してください。
```

### パターン7: Gitコミットメッセージ生成

```
AgentHooksを設定して、ファイルを保存したときに、
変更内容に基づいてGitコミットメッセージの案を生成してください。
```

### パターン8: TODOコメント検出

```
AgentHooksを設定して、ファイルを保存したときに、
TODOコメントがあれば、一覧を表示してください。
```

## 💡 使用上の注意

### クレジット消費を抑えるコツ

1. **対象ファイルを限定**
```
AgentHooksを設定して、src/ディレクトリ内の*.jsファイルのみを対象にしてください。
```

2. **シンプルな処理**
```
AgentHooksを設定して、ファイル保存時に簡単なログを記録するだけにしてください。
```

3. **手動トリガーを活用**
```
AgentHooksを設定して、自動実行ではなく、手動でトリガーできるようにしてください。
```

### 安全な体験方法

1. **短時間だけ有効化**
   - Hookを作成
   - 2-3回だけ動作を確認
   - すぐに無効化または削除

2. **シンプルなHookから始める**
   - 最初は簡単な処理（ログ記録など）
   - 慣れてから複雑な処理を試す

3. **クレジット残高を確認**
   - 体験前に残高を確認
   - 体験後に消費量を確認
