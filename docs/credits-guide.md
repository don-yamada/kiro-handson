# クレジットガイド

Kiroはクレジット制を採用しており、使用量に応じてクレジットが消費されます。このガイドでは、クレジットの仕組みと効率的な使い方を説明します。

**注意**: この情報は2026年2月時点のものです。最新情報は[Kiro公式サイト](https://kiro.ai/pricing)をご確認ください。

## 📊 クレジット制の仕組み

### クレジットとは

クレジットは、Kiroの使用量を測る単位です。Kiroに指示を出すたびに、処理の複雑さに応じてクレジットが消費されます。

### クレジット消費のタイミング

- Kiroにメッセージを送信したとき
- Kiroがコードを生成・編集したとき
- Kiroがファイルを読み込んだとき
- AgentHooksが自動実行されたとき

## 🎁 トライアルクレジット

### 内容

新規登録時に、約100クレジットのトライアルクレジットが付与されます（2026年2月時点）。

### 有効期限

トライアルクレジットには通常、有効期限があります。詳細はKiroの設定画面で確認してください。

### 使い切った後

トライアルクレジットを使い切った後も、Kiroは使用できますが、有料プランへのアップグレードが必要になります。

## 💰 各機能のクレジット消費目安

### 低コスト（1-3クレジット程度）

- **チャット（Vibe）**: 簡単な質問や会話
- **ファイル閲覧**: ファイルの内容を表示
- **簡単な説明**: 用語の説明、概念の解説

### 中コスト（5-10クレジット程度）

- **コード生成**: 関数やクラスの作成
- **コード編集**: リファクタリング、バグ修正
- **ファイル検索**: プロジェクト内の検索

### 高コスト（10-20クレジット程度）

- **Spec作成**: Requirements-first Workflowによる設計書生成
- **複雑なコード生成**: 複数ファイルにまたがる実装
- **大規模なリファクタリング**: プロジェクト全体の構造変更

### 非常に高コスト（20クレジット以上）

- **AgentHooks自動実行**: ファイル保存のたびに実行される処理
- **MCP連携**: 外部サービスとの大量データ処理
- **プロジェクト全体の分析**: 大規模なコードベースの解析

## 📈 有料プランの比較

### Pro Tier（約$20/月）

- **月間クレジット**: 約1,000クレジット
- **推奨用途**: 個人開発者、小規模プロジェクト
- **特徴**:
  - 基本的なSpec駆動開発
  - コード生成とリファクタリング
  - 限定的なAgentHooks使用

### Pro+ Tier（約$40/月）

- **月間クレジット**: 約2,000-3,000クレジット
- **推奨用途**: チーム開発、中規模プロジェクト
- **特徴**:
  - 無制限のSpec駆動開発
  - 高度なAgentHooks
  - MCP連携の本格活用
  - 優先サポート

### Enterprise（カスタム料金）

- **月間クレジット**: カスタム
- **推奨用途**: 大規模組織、エンタープライズ
- **特徴**:
  - 専用サポート
  - オンプレミス対応
  - カスタムMCPサーバー
  - SLA保証

## 💡 クレジット節約のコツ

### 1. 明確な指示を出す

❌ 悪い例:
```
コードを書いて
```

✅ 良い例:
```
src/utils.tsに、文字列を反転させるreverseString関数を作成してください。
引数はstring型、戻り値もstring型です。
```

明確な指示により、Kiroが一度で正しいコードを生成でき、再生成の無駄を防げます。

### 2. 段階的に進める

一度に大量の変更を依頼するのではなく、小さな単位で確認しながら進めましょう。

### 3. ファイルを絞り込む

不要なファイルを開いたままにせず、必要なファイルだけをKiroに見せましょう。

### 4. AgentHooksは慎重に

AgentHooksは便利ですが、ファイル保存のたびにクレジットを消費します。必要なときだけ有効化しましょう。

### 5. トライアル期間中の優先順位

トライアルクレジットでは、以下の順で体験することをおすすめします：

1. **レッスン1**: Spec駆動開発（25-35クレジット）
2. **レッスン2**: コード生成とリファクタリング（30-40クレジット）
3. **レッスン3**: AgentHooks（短時間のみ体験）
4. **レッスン4**: MCP連携（基本機能のみ）

**プラン1: 基本機能重視（約55-75クレジット）**

- レッスン3のAgentHooksは短時間のみ有効化
- 有料プランへのアップグレードを検討

## 🔄 クレジット残高の確認方法

1. Kiroの右上にあるアカウントアイコンをクリック
2. 「Settings」→「Billing」を選択
3. 現在のクレジット残高と使用履歴が表示されます

## 💳 有料プランへのアップグレード

### アップグレード方法

1. Kiroの設定画面を開く
2. 「Billing」タブを選択
3. 「Upgrade」ボタンをクリック
4. プランを選択して支払い情報を入力

### 支払い方法

- クレジットカード（Visa、Mastercard、American Express）
- PayPal
- 企業向け請求書払い（Enterpriseプランのみ）

## 🆘 クレジット不足時の対処法

### 症状

- 「クレジットが不足しています」というメッセージが表示される
- Kiroが応答しなくなる

### 対処法

1. **クレジット残高を確認**: 設定画面で残高をチェック
2. **有料プランにアップグレード**: 継続して使用する場合
3. **クレジット追加購入**: 一時的に追加購入（プランによる）
4. **翌月まで待つ**: 有料プランの場合、翌月にクレジットがリセット
