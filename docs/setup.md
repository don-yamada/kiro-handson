## 1. アカウント登録とトライアルクレジット

### アカウント登録

1. Kiroを起動すると、ログイン画面が表示されます
2. 「Sign Up」または「新規登録」をクリック
3. メールアドレスとパスワードを入力
4. 確認メールが届くので、リンクをクリックして認証

### トライアルクレジットの確認

新規登録すると、約100クレジットのトライアルクレジットが自動的に付与されます（2026年2月時点）。

**クレジット残高の確認方法**:
1. Kiroの右上にあるアカウントアイコンをクリック
2. 「Settings」または「設定」を選択
3. 「Billing」または「料金」タブを開く
4. 現在のクレジット残高が表示されます

💡 **ヒント**: トライアルクレジットは、レッスン1-2を完了するのに十分な量です。

**Kiroのインストールがまだの方**: [Kiro公式サイト](https://kiro.ai)からダウンロードしてください。

## 2. プロジェクトのクローン

### Gitを使用する場合

ターミナルまたはコマンドプロンプトで以下を実行：

```bash
git clone https://github.com/[your-username]/kiro-hands-on-tutorial.git
cd kiro-hands-on-tutorial
```

### Gitを使用しない場合

1. [GitHubリポジトリ](https://github.com/[your-username]/kiro-hands-on-tutorial)にアクセス
2. 「Code」ボタンをクリック
3. 「Download ZIP」を選択
4. ダウンロードしたZIPファイルを解凍
5. 解凍したフォルダを任意の場所に配置

## 4. 初期設定の確認

### プロジェクトをKiroで開く

1. Kiroを起動
2. 「File」→「Open Folder」（または「ファイル」→「フォルダを開く」）
3. クローンした`kiro-hands-on-tutorial`フォルダを選択
4. 「Open」をクリック

### ファイル構造の確認

左側のエクスプローラーに以下のファイル構造が表示されることを確認：

```
kiro-hands-on-tutorial/
├── README.md
├── docs/
│   ├── setup.md (このファイル)
│   ├── credits-guide.md
│   └── troubleshooting.md
├── lessons/
│   ├── lesson-01/
│   ├── lesson-02/
│   ├── lesson-03/
│   └── lesson-04/
└── .kiro/
    └── steering.md
```

## 4. 動作確認

### Kiroとの最初の対話

1. Kiroのチャット入力欄（通常は画面下部）を見つけます
2. 以下のメッセージを入力してEnterキーを押します：

```
こんにちは！動作確認のため、このプロジェクトの目的を教えてください。
```

3. Kiroが応答すれば、正常に動作しています

### ステアリングファイルの確認

ステアリングファイルが正しく読み込まれているか確認します：

1. Kiroに以下のように話しかけます：

```
.kiro/steering.mdファイルを開いてください。
```

2. ファイルが開かれ、内容が表示されれば成功です

## ✅ セットアップ完了！

すべての手順が完了したら、チュートリアルを開始できます。

次のステップ：
- [README.md](../README.md)に戻って、レッスン一覧を確認
- [lessons/lesson-01/README.md](../lessons/lesson-01/README.md)を開いて、レッスン1を開始

## 🆘 トラブルシューティング

セットアップ中に問題が発生した場合は、[troubleshooting.md](troubleshooting.md)を参照してください。

よくある問題：
- アカウント登録ができない → メールアドレスが正しいか確認
- クレジットが表示されない → ログアウトして再ログイン
- プロジェクトが開けない → フォルダのパスに日本語や特殊文字が含まれていないか確認
