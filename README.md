# Kiroハンズオンチュートリアル

Kiroの基本機能を体験できる対話型ハンズオンチュートリアルへようこそ！

このチュートリアルでは、Kiroとの対話を通じて、AI駆動開発の主要機能を実践的に学習できます。

## 🚀 始め方

1. このリポジトリをクローンして、Kiroで開きます
2. 進捗状況は [PROGRESS.md](PROGRESS.md) で確認できます
3. Kiroに以下のように話しかけるだけ：

```
レッスン1を始めます
```

たったこれだけ！あとはKiroが対話形式で案内してくれます。

## 📚 学べること

- Spec駆動開発（自然言語から設計書とコードを生成）
- ステアリングファイル（Kiroの性格カスタマイズ）
- コード生成とリファクタリング
- AgentHooks（自動化）
- MCP連携（外部ツール統合）

## 💡 使い方

各レッスンを始めるときは、Kiroに話しかけるだけ：

```
レッスン1を始めます
```
```
レッスン2を始めます
```
```
レッスン3を始めます
```
```
レッスン4を始めます
```
